GTA5线上小助手 - 常见问题解决办法

发表于2022-02-06|更新于2022-06-11|gta5
|字数总计:994|阅读时长:3分钟|阅读量:

提示

  • 再排查问题前，首先确保你使用的是最新版本小助手，并安装了.NET6必装运行库

https://dotnet.microsoft.com/zh-cn/download/dotnet/thank-you/runtime-desktop-6.0.10-windows-x64-installer

杀毒拦截导致出现错误提示

  • 问题分析：

杀毒拦截，软件缺少必要的权限

  • 解决办法：

关闭杀毒软件或者加入白名单，重启新打开小助手

  • 问题截图

此应用无法无法在你的电脑上运行

  • 问题分析：

小助手下载文件不完整导致的，也就是说小助手在下载过程中出现了损坏

  • 解决办法：

重新下载一次小助手即可

  • 问题截图

小助手UI错乱、UI渲染异常

  • 问题分析：

因为最近外星人音频管理程序更新导致的，NahimicOSD是一个用于在应用程序最终渲染结果上叠加另一个显示层的库，这个库会在应用程序运行时注入进程（NahimicOSD.dll），并在目标进程调用 DirectX 渲染时将叠加层加入渲染。

大佬的详细分析博客：
https://blog.walterlv.com/post/wpf-renders-wrong-because-of-nahimicosd.html

  • 解决办法：

到C盘找到这个路径

C:\ProgramData\A-Volute\DellInc.AlienwareSoundCenter\Modules\ScheduledModules\x64\

将 NahimicOSD.dll 文件重命名为 NahimicOSD备份.dll 即可，也就是下面这样，然后重启电脑即可解决

C:\ProgramData\A-Volute\DellInc.AlienwareSoundCenter\Modules\ScheduledModules\x64\NahimicOSD.dll

C:\ProgramData\A-Volute\DellInc.AlienwareSoundCenter\Modules\ScheduledModules\x64\NahimicOSD备份.dll

  • 问题截图

打开小助手提示 To run this application, you must install .NET Desktop Runtime 6.0.1(x64)

  • 问题分析：

缺少必要的运行库或者运行库安装不准确（这个是必装的）

  • 解决办法：

安装.NET 6.0.10 Desktop Runtime运行库即可

https://dotnet.microsoft.com/zh-cn/download/dotnet/thank-you/runtime-desktop-6.0.10-windows-x64-installer

请安装 Run desktop apps 这个x64运行库，不要安装 Run console apps 或者 Run server apps，否则你打开小助手仍然会提示你安装运行库

  • 问题截图

win7系统打开小助手未反应或者闪退

  • 问题分析：

由于小助手最新版本使用.NET6开发，win7系统需要额外安装最新补丁包才可以

  • 解决办法：

安装最新win7补丁包，该补丁包由俄罗斯大佬制作，微软原版win7补丁，可以放心安装
（该补丁包大概800mb左右，下载完成后安装即可，可能需要15~30分钟才能安装全部补丁）

https://blog.simplix.info/update7/

补丁包下载地址：

地址1：https://update7.simplix.info/UpdatePack7R2.exe
地址2：magnet:?xt=urn:btih:68FD0B5FE44FCD6F061F8C247C5CFDDC3E2142CA

补丁包安装截图

  • 问题截图

发生了未知错误 ‘T’ is an invalid start of a value.Path: $ | LineNumber: o lBytePositionInLine: 0.

  • 问题分析：

小助手打开时联网获取最新版本信息错误，这一般是无法访问网络导致的

  • 解决办法：

请检查网络，并运行小助手联网（或者直接无视这个提示，不影响小助手的使用）

  • 问题截图

无法启动此程序，因为计算机中丢失VCRUNTIME140.dll

  • 问题分析：

缺少必备运行库，一般是kiddion会出现这个提示

  • 解决办法：

安装“微软常用运行库合集”即可

https://crazyzhang.lanzoui.com/b04md71ve

  • 问题截图

小助手打开无反应，即使关闭杀毒

  • 问题分析：

一般用Edge下载小助手文件会出现这个问题，安全策略导致的

  • 解决办法：

右键小助手属性，勾选解除阻止

  • 问题截图

小助手能打开，但是功能都没有效果

  • 问题分析：

这一般是被杀毒拦截或者程序权限不足导致

  • 解决办法：

不要把小助手放到桌面，找个非系统盘（非C盘），建一个文件夹，比如D:\GTA5Help，然后把小助手放到该目录下使用
