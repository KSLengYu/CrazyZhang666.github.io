提问的智慧

提问的时候麻烦可以详细描述下出现问题的具体情况，这样大家也好帮你解决。

不要直接上来就是打不开、闪退，小助手里面有许多功能，第三方的和小助手自带的，如果是仅仅双击鼠标打开小助手程序是不可能导致闪退或出现问题的，主要是看你使用的具体功能，使用的哪一个辅助，这个辅助功能具体在哪一个位置。

如果描述混乱不清，这就是在浪费大家双方的时间，本来一个小小的问题几分钟就解决了，但是你描述不起或者是寥寥几笔，这个问题可能一个月都无法解决，因为大家都听不懂你的描述，无法帮助你

遇到问题尽量能有截图，但是本站批评论区是不能上传图片的，大家可以上传到图床，然后把图片链接发到评论区即可

下列图床可任选其一上传图片

必备运行库

最新发布日期 2022年5月10日
.NET 6.0 Desktop Runtime (v6.0.5)

小助手最新版本采用.NET6开发，所以需要安装对应的运行库才能正常使用，首先你要确保电脑上已经正确安装了运行库。

大家直接下载安装我下面链接的安装文件即可，不要自己去微软官网下载，否则你下载了错误的版本安装还是会打不开。

这个运行库有版本区分，如果你安装的6.0.1的版本，后续更新变成6.0.2版本，你就需要更新NET版本，否则可能会出现无法正常使用小助手的问题，所以大家要确认自己的运行库是最新版本。

这个运行库最新版本是6.0.10，所以无论你之前有没有安装，建议都先下载安装一遍

https://dotnet.microsoft.com/zh-cn/download/dotnet/thank-you/runtime-desktop-6.0.10-windows-x64-installer

小助手下载

如果你直接使用浏览器下载小助手，那么下载完毕一般都会被浏览器识别为病毒，但是你可以选择保留下载文件，这一般都是 Windows Defender安全中心 导致的，需要关闭 Microsoft Defender Smartscreen 选项

使用 Defender Control工具 来开/关 Windows Defender安全中心

https://www.sordum.org/9480/defender-control-v2-0/

杀毒报毒

我也不清楚为啥杀毒一直报毒，但是可以确认这是误报。大家可以暂时关闭杀毒软件或者是将小助手加入白名单，否则可能会出现各种奇奇怪怪的问题

小助手最新版本采用 .NET6 开发，并且完全开源，大家可以自行下载源码编译

如果你坚持认为小助手是病毒，请立刻删除从电脑上删除小助手所有相关文件，并以后不再使用

https://github.com/CrazyZhang666/GTA5OnlineTools

支持系统

小助手支持win7、win8、win8.1、win10、win11系统，也就是说可以玩GTA5的PC系统都是可以运行小助手的。

我这里比较推荐win10、win11系统，win7系统还是太老了，有时候总会有一些奇怪的问题，给自己的使用造成不必要的麻烦

win7系统说明

如果你是win7系统，最新版本小助手无法正常运行，请装最新win7补丁包即可解决。
（如果win7正常运行请忽略）

该补丁包由俄罗斯大佬制作，微软原版win7补丁，可以放心安装
（该补丁包大概800mb左右，下载完成后安装即可，可能需要15~30分钟才能安装全部补丁）

https://blog.simplix.info/update7/
